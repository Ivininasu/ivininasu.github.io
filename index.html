Project 1:
Arduino LED Memory Game

The basic overview of this project is that the four LEDs blink in a certain, random pattern and the player has to imitate the same pattern to level up, and eventually win, the game. If the player enters a pattern incorrectly, they lose the game and it starts a new pattern. Every time, you got it right, the game would give you the sequence again with an additional colour at the end.
